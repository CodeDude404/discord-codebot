# discord-codebot
A discord bot for all sorts of things.


## Music on voice call

### $join
Joins the runner's voice channel

### $leave
Leave's the runner's voice channel

### $music <play|list|stop|pause|resume> (song)
Alisies: $m

**play <song>**
Plays <song>
  
**list**
Shows a list of all avalile songs.
  
**stop**
Stops the music
  
**pause**
Pauses the music.

**resume**
Resumes the music.
